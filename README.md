API TESTING PROJECT 

This project contains a simple setup for testing REST api using postman.

The goal is to understand API endpoints,sending requests, receving responses, validating data.


PROJECT OVERVIEW:

This repository includes:

1> Basic API endpoints(GET,POST,PUT,DELETE)
2> POSTMAN API collection for testing
3> Node.js dummy server for API response (if included)
4> sample tets cases.
